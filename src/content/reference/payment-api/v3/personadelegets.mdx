##### **GET** /v3/persona/:hash/verify

Verify that a particular twin has a persona delegate.

##### Path Parameters

###### hash ***string***

##### Request Body Schema

```javascript
{
    "twin_id": 1234
}
```

##### Response Schema

```javascript
{
    "status": 200,
    "message": "OK",
    "hostname": "hostname_hash",
```
