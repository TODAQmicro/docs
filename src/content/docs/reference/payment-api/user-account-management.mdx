## User Account Management

## Make an Account
##### **POST** `/v2/account`

## Authorize an Account
##### **POST** `/v2/account/oauth/token`

## Make a User
##### **POST** `/v2/user`

## Authorize a User
##### **POST** `/v2/user/oauth/token`

## Get a User
##### **GET** `/v2/user/:id`

## Get Users
##### **GET** `/v2/users`
